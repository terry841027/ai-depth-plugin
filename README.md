# AI Depth Map Plugin

Real-time monocular depth estimation plugin for Resolume and compatible FFGL hosts, powered by Depth Anything V2 Small ONNX model.

## 功能說明

### 視覺化模式

| 模式 | 參數 | 說明 |
|------|------|------|
| Z-Depth 灰階 | `Scanline = OFF` | 將深度以灰階顯示（近=亮、遠=暗） |
| 掃描線特效 | `Scanline = ON` | 以掃描線疊加深度資訊，支援多種特效 |
| 粒子發光 | `Particle = ON` | 在前景輪廓邊緣生成發光粒子向外擴散 |

---

### 推理 / 效能控制

| 參數 | 預設值 | 範圍 | 說明 |
|------|--------|------|------|
| Quality | 0.5 | 0–1 | 品質/效能權衡，低值跳過幀以省資源，高值每幀推理 |
| Smooth | 0.35 | 0–1 | 時間平滑度（EMA alpha），低=反應靈敏，高=畫面穩定 |

---

### Z-Depth 深度視覺化

| 參數 | 預設值 | 範圍 | 說明 |
|------|--------|------|------|
| Invert | OFF | 布林 | 翻轉近/遠極性（原本近=亮，開啟後近=暗） |
| Depth Near | 0.0 | 0–1 | 深度範圍下限，用於重新映射可視深度區間 |
| Depth Far | 1.0 | 0–1 | 深度範圍上限，用於重新映射可視深度區間 |

---

### 全域掃描（深度切片）

| 參數 | 預設值 | 範圍 | 說明 |
|------|--------|------|------|
| Sweep | OFF | 布林 | 啟用動態深度切片，以亮帶標記特定深度層 |
| Sweep Direction | 0 | 布林 | 掃描方向：0 = 遠→近，1 = 近→遠 |
| Sweep Position | 0.0 | 0–1 | 切片位置，可與 BPM 同步驅動 |

---

### 掃描線特效

| 參數 | 預設值 | 範圍 | 說明 |
|------|--------|------|------|
| Scanline | OFF | 布林 | 切換 Z-Depth（關）與掃描線特效（開） |
| Effect | 0.4 | 0–1 | 特效類型：0–0.33 = 水平線、0.33–0.66 = 扭曲線、0.66–1.0 = 同心圓/點陣 |
| Density | 0.3 | 0–1 | 掃描線密度或點陣格數 |
| Width | 0.1 | 0–1 | 線條寬度或點半徑 |
| Warp | 0.3 | 0–1 | 深度位移量（僅扭曲線特效有效） |
| Offset | 0.0 | 0–1 | 掃描線捲動位置，可與 BPM 同步 |
| Blend | 0.5 | 0–1 | 混合模式：0 = 特效疊黑底，1 = 特效疊原始影像 |

---

### 粒子發光模式

| 參數 | 預設值 | 範圍 | 說明 |
|------|--------|------|------|
| Particle | OFF | 布林 | 啟用粒子發光模式 |
| P. Density | 0.5 | 0–1 | 格點數量（15–95 格），控制粒子分布密度 |
| P. Size | 0.5 | 0–1 | 粒子半徑大小 |
| P. Glow | 0.7 | 0–1 | 發光亮度倍率 |
| P. Drift | 0.3 | 0–1 | 粒子漂移速度與最大擴散距離 |

---

### 特效類型速查（Effect 參數）

| Effect 值 | 特效名稱 | 說明 |
|-----------|----------|------|
| 0.00–0.33 | 水平掃描線 | 固定間距水平線，密度與寬度可調 |
| 0.33–0.66 | 深度扭曲線 | 線條依深度值偏移產生波浪視覺 |
| 0.66–0.99 | 同心圓線 | 從畫面中心向外放射的環形線條 |
| ~1.00 | LED 點陣 | 格點發光，顏色從原始影像取樣 |
